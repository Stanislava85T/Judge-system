🔵 Cloud режим (5G/интернет) — Supabase
Създай проект на supabase.com и изпълни SQL-а по-долу, след което въведи URL и ключа.
📋 SQL за Supabase (кликни за да видиш) 
create table if not exists agg_scores (
  id text primary key default gen_random_uuid()::text,
  team_id text not null,
  judge_key text not null,
  value float not null,
  comp_key text default 'default',
  updated_at timestamptz default now(),
  unique(team_id, judge_key, comp_key)
);
create table if not exists agg_state (
  key text primary key,
  data jsonb,
  updated_at timestamptz default now()
);
alter table agg_scores enable row level security;
alter table agg_state enable row level security;
create policy "public rw scores" on agg_scores for all using (true) with check (true);
create policy "public rw state" on agg_state for all using (true) with check (true);
Как работи: Запази данните като .json файл на компютъра. На друго устройство — отвори agg-admin.html и зареди същия файл. Всички отбори, оценки и стартов ред ще бъдат възстановени.
